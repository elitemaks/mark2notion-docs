## Overview

The convert endpoint transforms Markdown text into Notion block objects that can be used with the Notion API or our append endpoint.

## Request

<ParamField header="x-api-key" type="string" required>
  Your Mark2Notion API key
</ParamField>

<ParamField body="markdown" type="string" required>
  The Markdown content to convert to Notion blocks
</ParamField>

## Response

<ResponseField name="status" type="string">
  Will be "success" for successful requests
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="data">
    <ResponseField name="blocks" type="array">
      Array of Notion block objects ready for use with Notion API
    </ResponseField>
    <ResponseField name="blockCount" type="number">
      Total number of blocks generated
    </ResponseField>
  </Expandable>
</ResponseField>

## Examples

<CodeGroup>

```bash curl
curl -X POST "https://api.mark2notion.com/api/convert" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "markdown": "# My Heading\n\nThis is a **bold** paragraph with a [link](https://example.com).\n\n- List item 1\n- List item 2"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.mark2notion.com/api/convert', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    markdown: `# My Heading

This is a **bold** paragraph with a [link](https://example.com).

- List item 1
- List item 2`
  })
});

const data = await response.json();
console.log(data.data.blocks);
```

```python Python
import requests

response = requests.post(
    'https://api.mark2notion.com/api/convert',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY'
    },
    json={
        'markdown': '''# My Heading

This is a **bold** paragraph with a [link](https://example.com).

- List item 1
- List item 2'''
    }
)

data = response.json()
print(data['data']['blocks'])
```

## Supported Markdown

The convert endpoint supports all standard Markdown syntax:

### Text Formatting
- **Bold text** (`**bold**`)
- *Italic text* (`*italic*`)
- ~~Strikethrough~~ (`~~strikethrough~~`)
- `Inline code` (`` `code` ``)

### Headings
- `# Heading 1`
- `## Heading 2`
- `### Heading 3`

### Lists
- Bulleted lists (`- item`)
- Numbered lists (`1. item`)
- Nested lists (indented)

### Links and Media
- [Links](https://example.com) (`[text](url)`)
- Images (`![alt](url)`)

### Code
- Inline code (`` `code` ``)
- Code blocks (``` ```language ```)

### Other
- Blockquotes (`> quote`)
- Horizontal rules (`---`)
- Tables (GitHub-flavored markdown)

## Usage Notes

<Info>
Each convert request counts as 1 API call against your quota, regardless of the markdown size.
</Info>

<Tip>
The returned blocks are standard Notion API block objects and can be used directly with the Notion API or passed to our append endpoint.
</Tip>

<Warning>
Very large markdown documents may take longer to process. Consider chunking large content for better performance.
</Warning>
